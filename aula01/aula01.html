Podemos inserir o JavaScript dentro do HTML de forma interna - no próṕrio código HTML, ou de forma externa - incluindo um arquivo externo.

Inserindo o JavaScript dentro do HTML na tag head:

Dentro do nosso <head> abrimos a tag <script> e inserimos o nosso código lá.

Colocando o JavaScript de forma externa:

Ao colocar ele de forma externa, podemos colocar ele no final do arquivo html com a tag: <script src=caminho-do-arquivo.js">. Isso faz com que a página seja carregada primeiro e só depois o arquivo js é renderizado.

Também podemos colocar ele dentro da tag head, mas ele será inicializado antes do resto da página, o que pode gerar bugs. Para evitar bugs e deixar o código mais organizado, deixando o link de arquivos dentro do head, podemos adicionar o atributo defer dentro da tag. Ficaria assim: <script src='caminho-js' defer>
